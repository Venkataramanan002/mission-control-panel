# Smooth Text Control Panel

This project is a Vite + React + TypeScript dashboard experience themed for the Smooth Text Control Panel. It ships with Tailwind CSS and shadcn/ui primitives.

## Getting started

```bash
npm install
npm run dev
```

## Building for production

```bash
npm run build
```

Preview the optimized build with:

```bash
npm run preview
```

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui
